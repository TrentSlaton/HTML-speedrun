What are some things that you really ought to know?

Can you create custom HTML Data Attributes? How? Show an example.

Yes. You prefix the attribute with "data-". Here's an example:
<a> href=""http://example.com" data-visited="08-13-22"</a>

What are HTML "entities" and what are some examples?

HTML entities allow you to display a unicode character without using the character itself. You use them with the ampersand character:

  • &#<number> where <number> is the character's codepoint in decimal
  • &#x<number> where <number> is the character's codepoint in hexadecimal
  • &#<name> where <name> is the named entity (not all entities have names)
